# Hardware

这里记录 SparkBot 的硬件复刻过程，包括 PCB、BOM、外壳、焊接顺序、首次上电检查和排障。

## 目录规划

```text
hardware/
├── pcb/            # PCB 资料与下单记录
├── enclosure/      # 外壳与打印设置
├── assembly/       # 装配步骤
└── bringup/        # 首次上电、测量、排障
```

## 首次上电原则

1. 先测短路，不要直接插电。
2. 先验证电源轨，再连接高价值模块。
3. 有异常发热立刻断电。
4. 每完成一个模块都记录测量结果。
