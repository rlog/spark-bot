# Firmware

这里放 SparkBot 的 ESP-IDF 固件相关内容。

## 计划

- 保留官方/复刻固件的可运行版本
- 新增自己的实验固件
- 逐步抽象显示、音频、摄像头、运动、网络、AI 配置

## 目录规划

```text
firmware/
├── upstream/       # 官方或复刻源码镜像/引用说明
├── experiments/    # 小实验：屏幕、音频、摄像头、Wi-Fi、运动
├── sparkbot-os/    # 自定义固件主工程
└── notes/          # 编译、烧录、排障记录
```
