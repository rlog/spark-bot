# 增强版 BOM（V1）

本文档帮助理解各模块的重要性与采购顺序。

| 模块 | 优先级 | 风险 |
|------|--------|------|
| 电源系统 | P0 | 高 |
| ESP32-S3 主控 | P0 | 低 |
| LCD 屏幕 | P0 | 中 |
| 摄像头 | P1 | 高 |
| 音频系统 | P1 | 高 |
| IMU | P2 | 高 |
| 电机 | P2 | 中 |
| 外壳 | P3 | 低 |

## 电源系统

目标：先验证供电，再连接其它模块。

首次检查：

- USB 5V
- 3.3V
- GND 是否短路
- 是否有异常发热

## ESP32-S3

负责 WiFi、USB、LCD、Camera、Audio。

学习重点：ESP-IDF、FreeRTOS、SPI、I2S。

## LCD

首次目标：点亮背光并成功刷新第一帧。

## 摄像头

首次只验证初始化和采集一帧图像，不做识别。

## 音频

先验证录音和播放，再接 AI。

## 采购原则

1. 不一次买齐。
2. 先确认 PCB 版本。
3. 高风险器件逐项核对。
4. 所有购买链接记录到仓库。
