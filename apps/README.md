# Apps

这里放 SparkBot 的桌面端、Web 控制台、调试工具。

## 想象中的能力

- 查看机器人在线状态
- 查看串口/网络日志
- 配置模型 API Key 和服务地址
- 推送消息给机器人播报
- 显示 GitHub、AI Coding Agent、天气、日程等状态

## 候选技术栈

- Web: React / Next.js / Vite
- UI: shadcn/ui / Ant Design
- 通信: WebSocket / MQTT / HTTP
- 桌面端: Electron / Tauri

## 第一阶段目标

- [ ] 先做一个本地 Web 控制台原型
- [ ] 能显示机器人连接状态
- [ ] 能发送一条测试消息给机器人
